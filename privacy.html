隐私政策

最后更新：2026 年 7 月 9 日

概述

双语 AI 翻译（以下简称"本扩展"）是一款运行在浏览器中的翻译工具。 我们非常重视您的隐私。本隐私政策旨在透明地说明本扩展如何处理您的数据。

核心原则：本扩展不收集任何个人数据。

数据处理说明

1. API Key（密钥）

您在使用本扩展时需要自行配置 AI 模型的 API Key。该 Key 仅通过 Chrome 的 chrome.storage.sync API 存储在您的本地浏览器中， 不会传输到任何第三方服务器（除了您指定的 AI 接口地址本身）。

2. 翻译请求

当您浏览网页时，本扩展会将页面中的文本发送到您自己配置的 AI 模型接口地址 （如 DeepSeek、Qwen、GLM 等）进行翻译。这些请求：

  • 直接发送到您指定的 API 地址，不经任何中间服务器中转
  • 仅包含当前页面需要翻译的文本片段
  • 不包含您的任何个人信息（如浏览器历史、Cookie 等）

3. 翻译缓存

为节省 API 调用次数，翻译结果会缓存在 Chrome 的 chrome.storage.local 中，仅保存在您的本地设备上。 您可以随时通过扩展弹窗的"清除所有缓存"按钮删除这些缓存。

4. 设置数据

您的扩展设置（模型选择、语言偏好、显示模式等）通过 Chrome 的同步存储 （chrome.storage.sync）保存，仅存储在您的浏览器中。 如果您登录了 Google 账号，这些设置可能会通过 Chrome 同步功能在多台设备之间同步。

数据收集声明

本扩展绝不收集、上传、分享或出售任何用户数据。 所有配置信息、API Key 和翻译缓存均存储在用户本地，不会传输到我们或任何第三方的服务器。

第三方服务

本扩展本身不依赖任何第三方分析服务或外部 API（除用户自行配置的 AI 模型接口外）。 您选择的 AI 模型提供商（如 DeepSeek、阿里云、智谱等）可能有其独立的隐私政策， 建议您在使用前查阅相关服务条款。

权限说明

  • storage：用于保存您的设置和翻译缓存，所有数据均在本地。
  • unlimitedStorage：翻译缓存可能需要较大存储空间，避免达到默认存储上限。
  • <all_urls>：用于在您访问的网页上执行翻译功能， 仅在页面中查找文本并添加翻译结果，不会读取或修改您的个人信息。
