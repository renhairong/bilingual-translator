🌐 双语翻译 · iOS 版

在 iPhone 的 Safari 浏览器中一键翻译网页

📌 安装步骤

  1. 将此页添加到 Safari 阅读列表或书签，方便日后访问
  2. 复制下面的书签代码
  3. 在 Safari 中创建一个新书签（随便填个名称）
  4. 编辑该书签，将 URL 替换为复制的代码
  5. 打开任意外文网页，点击这个书签即可翻译

📋 书签代码

长按或右键复制下面整段代码：

javascript:void(function(){var s=document.createElement('script');s.src='https://'+location.host+'/bookmarklet.js?'+Date.now();document.body.appendChild(s)})();
已复制 ✓
💡 提示：第一次使用时，会弹出配置窗口。 填入你的 API Base URL、API Key 和 Model 名称即可
（与电脑端 Chrome 扩展的设置相同）

⚙️ 配置说明

字段说明
API Base URL你的 AI 接口地址，如 https://api.deepseek.com/v1
API Key你的 API 密钥
Model模型名称，如 deepseek-chat

配置信息仅保存在当前网站的 localStorage 中，不会上传。

⚠️ 隐私说明

  • API Key 存储在当前浏览网页的 localStorage 中，每次翻译时发送到你的 AI 接口
  • 本页面部署在 Vercel 上，仅作为请求中转，不存储任何用户数据
  • 页面文本通过云函数转发到你自己配置的 AI 接口
